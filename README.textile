h2. INSTALLATION

<pre>
<code>
git clone git://github.com/dcrec1/rspec-maven-plugin.git
cd rspec-maven-plugin
mvn install
</code>
</pre>

h2. CONFIGURATION

Just put this in your pom.xml:

<pre>
<code>
<plugin>
    <groupId>org.codehaus.mojo</groupId>
    <artifactId>rspec-maven-plugin</artifactId>
    <executions>
        <execution>
            <id>test</id>
            <phase>test</phase>
            <goals>
                <goal>spec</goal>
            </goals>
        </execution>
    </executions>
</plugin>
</code>
</pre>
